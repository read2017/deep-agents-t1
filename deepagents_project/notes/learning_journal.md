## Deep Agents

**简短讲解**

Deep Agents 可以理解为一种“更会做事的代理框架”，不是单个模型本身。  
它的核心是把复杂任务拆成几个能力一起协作：

- **规划**：先分解任务，不是直接乱做
- **工具使用**：会读文件、搜索、编辑、调用外部能力
- **委派**：把独立子任务交给 subagent 处理
- **上下文管理**：避免所有细节都塞进一个对话里，减少混乱和上下文爆炸

一句话：**Deep Agents = 模型 + 任务分解 + 工具 + 子代理协作 + 上下文控制**。

## Deep Agents

**简短讲解**

Deep Agents 可以理解为：**一套让 AI 更稳定完成复杂任务的“代理执行框架”**，不是单纯的大模型本身。  
它通常包含 4 个关键能力：

1. **规划**：把大任务拆成小步骤  
2. **工具使用**：读文件、搜索、编辑、调用外部能力  
3. **委派**：把独立子任务交给 subagent  
4. **上下文管理**：避免主线程被过多细节塞满

一句话：**Deep Agents = 模型 + 任务拆解 + 工具 + 子代理协作 + 过程管理。**
